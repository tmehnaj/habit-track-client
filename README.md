#  GameHub: The Ultimate Gaming Resources

## Project Name
**GameHub**

---

## Purpose
GameHub is a full package for **game lovers**. Anyone can **download games** and use games with it's complete details and ratings.

---

## Live URL
https://gaming-hub-ae400.web.app/


---

## Key Features
* **Game Discovery:** Browse and search many categories of games.
* **Detail For Every Game** View **Rating** , see **Game Description**, also **Download Link**

* **Game Profile** Create your **Game Profile** for getting exciting surprises and  **gift     card**
* **Responsive Design:** Fully responsive for every size devices.

---

## NPM Packages Used
The following npm packages were instrumental in developing this project:

* **react:** For building the user interface.
* **firebase:** For implementing smooth authentications.
* **react-router-dom:** For client-side routing and navigation.
* **styled-components** (or **tailwind-css**): For component-based styling.
* **styled-components** (or **daisy ui**): For ready UI components.
* **styled-components** (or **motion**): Libraries for smooth animations.
